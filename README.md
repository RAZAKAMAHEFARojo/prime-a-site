# Prime A

Site vitrine React + Vite + Tailwind CSS pour Prime A, cabinet de conseil en performance, finance, organisation et solutions digitales.

## Commandes

```bash
npm install
npm run dev
```

Build de production :

```bash
npm run build
npm run preview
```

## Stack

- React 19
- Vite 8
- Tailwind CSS 4 via `@tailwindcss/vite`
- React Router 7
